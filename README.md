🌎 English
> Extracted from a real-world SaaS system.

### What this is

An operational dashboard that simulates day-to-day management of a sports club, focused on business state and pending actions.

---

### Problem it solves

In many systems, key information is fragmented:

- Enrollments in one place
- Payments in another
- Pending actions without visibility

This demo centralizes everything into a single actionable view.

---

### What you can do in this demo

- Detect overdue payments and act immediately
- Identify pending enrollments
- Filter data by time range (today, 7 days, 30 days)
- Mark payments as resolved
- Simulate client follow-up (e.g. WhatsApp contact)

---

### What it includes

- KPI aggregation from structured data
- Business logic (payment status, active students, pending cases)
- Action-driven UI, not just static metrics
- Time-based filtering system
- Clean separation of logic in src/utils/metrics.js

---

### How to run

npm install
npm run dev

## 🇪🇸 Español
> Extraído de un sistema SaaS real en producción.

### Qué es

Dashboard operativo que simula la gestión diaria de un club deportivo, enfocado en estado del negocio y acciones pendientes.

---

### Problema que resuelve

En muchos sistemas, la información clave está dispersa:

- inscripciones por un lado
- pagos por otro
- pendientes sin seguimiento

Esto genera fricción operativa.

Este dashboard centraliza todo en una sola vista clara y accionable.

---

### Qué puedes hacer en esta demo

- Detectar pagos vencidos y actuar inmediatamente
- Identificar inscripciones pendientes
- Filtrar datos por rango de tiempo (`hoy`, `7 días`, `30 días`)
- Marcar pagos como resueltos
- Simular seguimiento a clientes (ej: contacto por WhatsApp)

---

### Qué incluye

- Agregación de KPIs a partir de datos estructurados
- Lógica de negocio (estado de pagos, alumnos activos, pendientes)
- UI orientada a acciones, no solo visualización
- Sistema de filtros dinámicos por tiempo
- Separación de lógica en `src/utils/metrics.js`

---

### Cómo ejecutarlo

npm install
npm run dev
